# lab-main-leihuogaming

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面存档与展示用途。

## 目录说明

仓库目录结构如下：

- `pages/` — 存放所有 HTML 页面文件
- `assets/` — 存放页面引用的资源文件（如 CSS、JS、图片等）
- `docs/` — 存放相关说明文档或补充资料
- `index.html` — 仓库首页入口（如有）

请根据实际需要调整目录结构。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档文件，彼此之间无依赖关系。每个页面均可直接通过浏览器打开查看，或部署至任意静态托管服务。

### 页面命名规范

- 页面文件使用英文或拼音命名，建议使用小写字母与连字符。
- 每个页面应保持自包含，尽量避免外部资源依赖（或确保资源路径正确）。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并更新本文件中的内容索引（如有）。
- 资源更新：如需修改共享资源，请确保不影响已有页面的正常显示。
- 删除页面：直接移除对应文件即可，无需额外操作。

本仓库的维护者会定期检查页面可访问性，但不对页面内容的时效性做保证。
